# Privacy Policy

**Effective Date:** 28 June, 2026

This Privacy Policy applies to the AI Timeline Chrome Extension ("the Extension").

## 1. Local Processing & No External Data Transmission
The Extension processes your data strictly locally on your device to provide its core functionality. **We do not transmit, distribute, sell, or store your personal data on any external servers.** 

## 2. Local Data Storage
To function correctly, the Extension saves specific user preferences and operational state data using your browser's local storage API (`chrome.storage`). 
*   **What is stored:** UI settings, saved prompts, custom created favorite folders, and saved requests/responses.
*   **Where it is stored:** This data remains strictly on your device and is never transmitted to third-party analytics services or developers.
*   **Data Deletion:** You can permanently delete this data at any time by clearing your browser's local storage or uninstalling the Extension.

## 3. Required Permissions and Data Handling
The Extension requires specific browser permissions to operate. All data accessed via these permissions is handled locally:
*   **`storage`:** Used to persistently save your bookmarked requests, responses, custom categories, user notes, and frequent prompt templates on your device.
*   **`contextMenus`:** Used to add custom right-click utility options ("Quote Selection" and "Copy without format") to your browser.
*   **`clipboardWrite`:** Used to write plain, style-free text to your system clipboard or as a fallback when inserting templates into the chat input.
*   **Host Permissions:** Used strictly to inject the navigation sidebar, bookmark buttons, and prompt helpers directly into the supported AI chat interfaces.

None of these permissions are used to track your browsing history or extract personal information for external use.

## 4. Chrome Web Store Limited Use Policy
The Extension's use and transfer of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

## 5. Changes to This Policy
If the data handling practices of the Extension change in the future, this policy will be updated, and significant changes will be communicated via the Chrome Web Store release notes.
